# Library Collection Development System (LCDS)

LCDS is a local-first bibliographic collection development application for importing, searching, analyzing, reporting on, and enriching large CSV datasets of books, films, television, animation, comics, manga, anime, video games, and related media.

## Phase 1 Objective

- CSV upload
- field mapping
- import preview
- SQLite storage
- searchable records
- advanced filters
- record detail pages
- overview dashboard
- reports and visualizations
- data issues panel

## Development Rule

Build in small, controlled slices. No AI automation in Phase 1.
